# 🧾 ระบบบริหารสำนักงานบัญชี Pamee

ระบบบริหารจัดการสำนักงานบัญชีครบวงจร - Prototype Version

## 🌐 Demo URL
**[กรุณาใส่ลิงก์หลัง deploy]**

## 📱 หน้าที่ใช้งานได้

| หน้า | ลิงก์ | คำอธิบาย |
|------|-------|----------|
| Dashboard | `/dashboard` | หน้าหลัก สรุปภาพรวม |
| ลูกค้า | `/clients` | รายชื่อลูกค้าทั้งหมด |
| รายละเอียดลูกค้า | `/clients/company-1` | ข้อมูลลูกค้ารายบริษัท |
| รายรับ-รายจ่าย | `/transactions` | บันทึกรายการเงิน |
| ใบแจ้งหนี้ | `/invoices` | ออกบิล/ติดตามการชำระ |
| OCR สแกนบิล | `/ocr` | อัปโหลดรูปบิล AI อ่านให้ |
| ยื่นภาษี | `/tax` | จัดการการยื่นภาษี |
| รายงาน | `/reports` | รายงานการเงิน |
| AI Chat | `/chat` | ถามตอบกับ AI |
| ตั้งค่า | `/settings` | ตั้งค่าระบบ |
| พนักงาน | `/staff` | จัดการพนักงาน (Admin) |

## 🛠 Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Zustand (State Management)

## 📝 หมายเหตุ

นี่คือ **Prototype** สำหรับทดสอบ UI/UX เท่านั้น
- ข้อมูลเป็น Mock Data (ไม่ใช่ข้อมูลจริง)
- ยังไม่มีระบบ Login จริง
- ยังไม่เชื่อมต่อ Database
